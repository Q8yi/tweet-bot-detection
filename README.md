# Bot Detection (In class Kaggle Competition)

This project aims to utilise machine learning model to predict whether a tweet is made by bot or human.



# Tech Stack:
Python pandas, nltk, demoji, sklearn, vaderSentiment, xgboost, spacy


```shell
npm install
pip3 install -r requirements.txt
```


# How it has been done:
1) Conducting EDA
2) Feature Engineering
3) Converting data to numerical data
4) Classification
