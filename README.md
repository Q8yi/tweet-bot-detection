# Bot Detection (In class Kaggle Competition)

This project aims to utilise machine learning model to predict whether a tweet is made by bot or human.



## Tech Stack:
**EDA:** Python Pandas 
**Feature Engineering:** Nltk, Demoji, Sklearn, VaderSentiment, Spacy 
**Machine Learning:** Xgboost, LightGBM, RandomForest Classifier
**Model Improvement**: Hyperparameter Tuning, 


```shell
npm install
pip3 install -r requirements.txt
```


## How it has been done:
1) Conducting EDA
2) Feature Engineering
3) Converting data to numerical data
4) Classification
